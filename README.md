# Computer Vision Mini Project – Handwritten Digit Classification

This project demonstrates a basic Computer Vision workflow for handwritten digit classification.

The goal is to classify digit images using pixel-based features and a machine learning classification model.

---

## Project Overview

In this project, I used the built-in digits dataset from Scikit-learn.  
Each image contains a handwritten digit from 0 to 9.

The image pixels are used as numerical features to train a classification model.

---

---

## Workflow

1. Load the handwritten digits dataset
2. Display sample digit images
3. Split the dataset into training and testing data
4. Train a Random Forest classification model
5. Evaluate the model using accuracy and classification report
6. Visualize performance using confusion matrix
7. Predict sample digit images
